linux commands

ctrl + alt + T -  команда открытие терминала 

win + пробел - это смена языка

~ - символ домашней директории 
/ - символ корневой дириктории 
. - обозначает текущую директорию 
.. - обозначает предыдущию директорию

1. ls - команда для вывода всех директории (папка) и файлов в текущей (рабочей) директории в терминале

2. cd (change directory) - комната для перехода из одной директории в другую 

3. mkdir -  создает директории 

4. touch - создает файл 

5. rmdir -  удаляет пустую директорию 

6. rm -rf -  удаляет вообще все (принудительное удаление) 

7. rm - удаляет файл

8. ls -a - показывает скрытые и обычные файлы директории
   ls -la -  покахывает подробную информацию
   
9. pwd(print working directory) - выводит путь до текущей директории

10. cp - копирует файл или директории указанное место 
	ср <название> <куда>
	
11. mv -  команда для перемещения файла или директории, а также для переименования файла или директории
	переименования: mv john.txt jamia.txt
	перемещение: mv john.txt directoru/
	
12. cat - показывает содержание файла

13. nano - открывает файл для редоктирование в графическом интерфейсе нано *если нет файла, то нано создаст новый 

14. echo - команда для быстрой записи текста в файл 
	echo текст > файл
